# 云旅无忧用户端 H5 原型

这是根据当前文件夹中的 40 张用户端 UI 参考图实现的本地静态前端原型。

## 打开方式

直接双击 `index.html`，或在浏览器中打开：

```text
./index.html
```

## 文件结构

```text
云旅无忧用户端代码实现/
├── index.html
├── styles.css
├── app.js
└── assets/
```

## 实现范围

- 覆盖参考图中的首页、目的地、活动、智能管家、智能设备、人工向导、紧急求助、订单、消息、我的、商城、健康、政策、设置等 40 个页面入口。
- 采用单页应用方式，通过左侧页面导航或移动端顶部横向导航切换页面。
- 主要界面结构、状态栏、顶部导航、底部 Tab、服务宫格、卡片列表、表单、时间线、设置开关、购物车结算条等均由 HTML/CSS/JS 代码渲染。
- `assets/` 中的图片从参考图裁切为 banner、商品图、头像、活动图等内容素材，不直接使用整页截图作为页面。

## 说明

图标使用在线 Lucide CDN。离线打开时页面布局仍可显示；联网打开时图标会自动渲染。
