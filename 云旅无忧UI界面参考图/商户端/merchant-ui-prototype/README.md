# 云旅无忧商户端 UI 代码实现

这是根据当前目录 70 张商户端 UI 参考图实现的静态前端原型。

## 使用方式

直接打开 `index.html`，或在当前文件夹启动本地服务：

```bash
python3 -m http.server 5187
```

然后访问：

```text
http://localhost:5187/
```

## 文件结构

- `index.html`：原型页面框架。
- `styles.css`：商户端视觉样式、移动端画布、卡片、表单、导航等组件样式。
- `app.js`：70 个页面的数据、页面切换和组件化渲染逻辑。
- `assets/`：从参考图裁出的 logo、服务图、头像、资质和门店图片资产。
