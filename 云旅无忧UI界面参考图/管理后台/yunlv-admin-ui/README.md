# 云旅无忧管理后台静态实现

这是根据当前文件夹中 53 张管理后台 UI 参考图实现的无依赖静态前端。

## 使用方式

直接打开 `index.html` 即可运行：

```text
yunlv-admin-ui/index.html
```

## 覆盖页面

- 登录页
- 数据概览与运营/老人/向导数据大屏
- 用户、老人、家属、健康档案、服务记录
- 人工向导、排班、审核、收入评价
- 商户、服务项目、服务新增、结算、资源与价格配置
- 订单、创建订单、订单详情、售后评价
- 任务调度、派单规则、调度详情
- 异常、设备异常、投诉、复盘
- 活动、Banner、政策、资讯、公告
- 设备绑定、设备详情、健康数据监测
- 账号、菜单权限、操作日志、系统配置、消息中心、AI知识库

## 技术说明

- 原生 HTML/CSS/JavaScript，无构建步骤、无外部依赖。
- 通过 URL hash 切换页面，例如 `#dashboard`、`#users`、`#ai-knowledge`。
- 图表、地图热力、排班表、权限树、规则编辑器等均用代码绘制或组合。
