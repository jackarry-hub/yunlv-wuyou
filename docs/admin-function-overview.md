# 7.1 管理后台功能总览

本页对应“7.1 管理后台功能总览”，已按 MVP 可演示版本落到后台真实接口、管理端首页和自动验收脚本。

## 功能总览

| 模块 | 功能需求 | 优先级 | 验收标准 |
| --- | --- | --- | --- |
| 首页数据概览 | 用户数、老人数量、在线设备、待处理订单、异常预警、今日服务 | P0 | 进入后台可看到关键指标。 |
| 老人/用户管理 | 用户档案、家属绑定、健康档案、紧急联系人、服务记录 | P0 | 可查询、查看、编辑基础信息。 |
| 人工向导管理 | 向导资料、审核、服务类型、在线状态、订单记录、评价、收入 | P0 | 可审核向导并查看状态。 |
| 商户管理 | 商户入驻、资质审核、服务上架、订单、评价、结算 | P0 | 商户审核和服务管理可用。 |
| 订单管理 | 全部订单、待派单、已派单、服务中、待确认、已完成、已取消 | P0 | 订单状态清晰，可人工干预。 |
| 任务调度 | 派单给人工向导或商户，支持按位置、服务类型、评分、空闲状态匹配 | P0 | 后台可手动派单和改派。 |
| 异常报告 | SOS、设备异常、摔倒检测、长时间未动、客户投诉、服务异常 | P0 | 异常可处理，记录处理结果。 |
| 活动管理 | 活动新增、编辑、上下架、报名管理、地图位置、分类 | P0 | 用户端活动地图可同步展示。 |
| 设备管理 | 设备列表、绑定用户、连接状态、电量、数据同步记录 | P1 | 可查看设备状态与模拟数据。 |
| 内容管理 | Banner、政策指南、旅居故事、通知公告 | P1 | 前端内容可配置。 |
| 数据大屏 | 老人监测大屏、人工向导调度大屏、服务数据统计 | P1 | 可展示核心统计和地图。 |
| 系统权限 | 角色、权限、菜单、操作日志 | P0 | 不同角色看到不同菜单。 |

## 工程实现

- 后台功能总览模块：`server/lib/admin-function-overview.js`
- 后台查询接口：`GET /api/admin/functions/overview`
- 管理端展示页：`/admin/#dashboard`
- 自动验收：`npm run test:admin-functions`

## 当前边界

- P0 模块已全部接入模拟数据库和后台 API。
- P1 模块为试运营可演示能力，设备、内容和数据大屏已能展示模拟数据。
- 后续接真实数据库时，保留接口结构，只替换数据读取层。
