# 云旅无忧 API 接口文档

> 版本：v1.0 | 基础地址：`http://localhost:5173` | 认证方式：Bearer Token (JWT)

---

## 一、全局说明

### 1.1 请求规范

- 基础路径：所有接口支持 `/api/*` 工程路径和裸路径两种方式
- Content-Type：`application/json; charset=utf-8`
- 认证头：`Authorization: Bearer <token>`（登录接口除外）
- 跨域：支持 CORS，`Access-Control-Allow-Origin: *`

### 1.2 响应格式

```json
// 成功
{ "success": true, "data": {}, "meta": {} }

// 失败
{ "success": false, "error": { "message": "错误信息", "details": {} } }
```

### 1.3 角色权限

| 角色 | 标识 | 权限数 |
|------|------|--------|
| 长者用户 | elder | 15 |
| 家属 | family | 15 |
| 人工向导 | guide | 8 |
| 商户 | merchant | 8 |
| 管理员 | admin | 28 |

## 二、认证接口

### POST /api/auth/login

账号登录，获取JWT Token。

**请求体：**
```json
{
  "phone": "13800005678",
  "role": "elder",
  "password": "任意值（MVP阶段不校验密码）"
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9...",
    "user": {
      "id": "user-001",
      "nickname": "李奶奶",
      "role": "elder",
      "avatar": "/user/assets/avatar-user.jpg",
      "permissions": ["user:read", "user:write", "order:create", ...]
    }
  }
}
```

### POST /api/auth/wechat-login

微信登录（MVP阶段模拟，使用code模拟获取用户信息）。

## 三、用户端接口

### GET /api/user/profile
获取当前用户资料。

### PUT /api/user/profile
更新用户资料。
```json
{ "nickname": "新昵称", "avatar": "头像URL", "phone": "手机号" }
```

### GET /api/elder/profile
获取老人健康档案。

### PUT /api/elder/profile
更新老人健康档案。
```json
{ "name": "姓名", "gender": "女", "age": 72, "healthTags": "高血压,糖尿病", "address": "地址" }
```

### GET /api/family-contacts
获取紧急联系人列表。

### POST /api/family-contacts
新增紧急联系人。
```json
{ "name": "张先生", "relation": "儿子", "phone": "13900001234", "isDefault": false }
```

### PUT /api/family-contacts/:id
编辑紧急联系人。

### DELETE /api/family-contacts/:id
删除紧急联系人。

### POST /api/family-contacts/:id/call
拨打紧急联系人（接口预留）。

## 四、AI管家接口

### POST /api/ai/chat
AI智能问答。
```json
{ "question": "附近有什么好玩的？", "context": "可选历史上下文" }
```
**响应：**
```json
{
  "success": true,
  "data": {
    "id": "chat-001",
    "answer": "AI回复内容...",
    "intent": "景点推荐",
    "recommendations": []
  }
}
```

### GET /api/ai/history
获取AI对话历史记录。

### GET /api/ai/quick-questions
获取快捷问题列表（≥7个预设问题）。

### POST /api/ai/quick-questions/:id/ask
使用快捷问题自动发起提问。

### POST /api/ai/voice/transcribe
语音识别结果转AI对话。
```json
{ "text": "语音识别文本", "duration": 3.5 }
```

### GET /api/ai/recommendations
获取服务推荐列表。

### GET /api/ai/service-records
获取服务咨询记录。

## 五、订单接口

### POST /api/orders
创建订单。
```json
{
  "serviceType": "陪诊服务",
  "providerType": "guide",
  "time": "2026-06-20 09:00",
  "location": "昆明市第一人民医院",
  "amount": 200,
  "note": "备注信息"
}
```

### GET /api/orders
查询订单列表。支持查询参数：`?status=待服务&page=1&limit=10`

### GET /api/orders/:id
查询订单详情（含时间线）。

### POST /api/orders/:id/cancel
取消订单。
```json
{ "reason": "取消原因" }
```
