## 二、快速部署

### 2.1 方式一：直接运行

```bash
# 1. 克隆代码
git clone <仓库地址> /opt/yunlv-wuyou
cd /opt/yunlv-wuyou

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入实际配置值

# 3. 启动服务
node server.js

# 服务将在 http://localhost:5173 启动
```

### 2.2 方式二：Docker 部署

```bash
# 1. 克隆代码
git clone <仓库地址> /opt/yunlv-wuyou
cd /opt/yunlv-wuyou

# 2. 构建镜像
docker build -t yunlv-wuyou:mvp .

# 3. 启动容器
docker run -d \
  --name yunlv-wuyou \
  -p 5173:5173 \
  -e YUNLV_AUTH_SECRET="your-long-random-secret" \
  -e DEEPSEEK_API_KEY="your-deepseek-key" \
  -v yunlv-runtime:/app/.runtime \
  yunlv-wuyou:mvp
```

### 2.3 方式三：Docker Compose

```bash
cd /opt/yunlv-wuyou

# 编辑环境变量
export YUNLV_AUTH_SECRET="your-long-random-secret"
export DEEPSEEK_API_KEY="your-deepseek-key"

# 启动
docker compose -f deploy/docker-compose.yml up -d

# 查看日志
docker compose -f deploy/docker-compose.yml logs -f

# 停止
docker compose -f deploy/docker-compose.yml down
```

### 2.4 方式四：Render Cloud

项目已配置 `render.yaml`，直接关联 Git 仓库即可自动部署：
- 服务类型：Web Service
- 构建命令：（无需构建）
- 启动命令：`node server.js`
- 端口：5173

## 三、环境变量配置

| 变量名 | 必填 | 默认值 | 说明 |
|--------|------|--------|------|
| PORT | 否 | 5173 | 服务监听端口 |
| NODE_ENV | 否 | production | 运行环境 |
| YUNLV_RUNTIME_DIR | 否 | .runtime | 运行时数据目录 |
| YUNLV_SEED_DB | 否 | data/mock-db.json | 初始种子数据路径 |
| YUNLV_AUTH_SECRET | **是** | （不安全默认值） | JWT签名密钥 |
| DEEPSEEK_API_KEY | 否 | （空） | DeepSeek AI API密钥 |
| DEEPSEEK_MODEL | 否 | deepseek-chat | AI模型名称 |
| DEEPSEEK_API_BASE | 否 | https://api.deepseek.com | AI API地址 |
| DB_CLIENT | 否 | json | 数据库类型(json/mysql) |
| DB_HOST | 条件 | 127.0.0.1 | MySQL主机（DB_CLIENT=mysql时必填） |
| DB_PORT | 条件 | 3306 | MySQL端口 |
| DB_NAME | 条件 | yunlv_wuyou | 数据库名 |
| DB_USER | 条件 | yunlv | 数据库用户 |
| DB_PASSWORD | 条件 | （空） | 数据库密码 |
| DB_POOL_SIZE | 否 | 10 | 连接池大小 |

## 四、域名与HTTPS配置

### 4.1 域名解析

```
A记录：yunlv-wuyou.com → 服务器IP
A记录：www.yunlv-wuyou.com → 服务器IP
A记录：admin.yunlv-wuyou.com → 服务器IP（可选）
```

### 4.2 Let's Encrypt 证书

```bash
# 安装 Certbot
apt update && apt install -y certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d yunlv-wuyou.com -d www.yunlv-wuyou.com

# 自动续期（已自动配置）
certbot renew --dry-run
```

### 4.3 Nginx 配置

将 `deliverables/06-nginx.conf` 复制到 Nginx 配置目录：

```bash
cp deliverables/06-nginx.conf /etc/nginx/sites-available/yunlv-wuyou.conf
ln -s /etc/nginx/sites-available/yunlv-wuyou.conf /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx
```

## 五、MySQL 数据库初始化

```bash
# 创建数据库和用户
mysql -u root -p <<EOF
CREATE DATABASE yunlv_wuyou CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'yunlv'@'%' IDENTIFIED BY 'your-secure-password';
GRANT ALL PRIVILEGES ON yunlv_wuyou.* TO 'yunlv'@'%';
FLUSH PRIVILEGES;
EOF

# 导入表结构
mysql -u yunlv -p yunlv_wuyou < database/schema.sql
mysql -u yunlv -p yunlv_wuyou < database/migrations/001_initial_mysql.sql
```

## 六、进程管理

### 使用 PM2（推荐）

```bash
# 安装 PM2
npm install -g pm2

# 启动服务
pm2 start server.js --name yunlv-wuyou

# 查看状态
pm2 status

# 查看日志
pm2 logs yunlv-wuyou

# 开机自启
pm2 startup
pm2 save

# 重启
pm2 restart yunlv-wuyou

# 停止
pm2 stop yunlv-wuyou
```

### 使用 Systemd

```ini
# /etc/systemd/system/yunlv-wuyou.service
[Unit]
Description=YunLv WuYou AI Platform
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/yunlv-wuyou
ExecStart=/usr/bin/node server.js
Restart=on-failure
RestartSec=10
EnvironmentFile=/opt/yunlv-wuyou/.env

[Install]
WantedBy=multi-user.target
```

```bash
systemctl daemon-reload
systemctl enable yunlv-wuyou
systemctl start yunlv-wuyou
systemctl status yunlv-wuyou
```
