## 三、自动化测试脚本

项目已包含46个自动化测试脚本，位于 `/scripts/` 目录：

```bash
# 运行全部测试
npm test

# 运行单项测试
npm run test:smoke              # 冒烟测试
npm run test:api-contract       # API契约测试
npm run test:acceptance-flow    # 验收流程测试
npm run test:business-flow      # 业务流程测试
npm run test:full-interop       # 全量互通测试
npm run test:mvp-api            # MVP接口测试
npm run test:minimum-delivery   # 最低交付测试
```
