# 云旅无忧 UI/组件规范

> 版本：v1.0 | 适用端：移动端（用户/向导/商户）+ 管理后台

---

## 一、设计基础

### 1.1 设计尺寸

| 端 | 设计稿宽度 | 单位 | 适配方案 |
|----|-----------|------|----------|
| 移动端（微信小程序/App） | 750rpx | rpx | uni-app 自动转换 |
| 管理后台 | 1440px | px | 响应式布局，最小支持 1280px |

### 1.2 色彩系统

| 色彩角色 | 色值 | 用途 |
|----------|------|------|
| 主色（品牌蓝） | #4A90D9 | 主要按钮、导航栏、选中态 |
| 辅色（温暖橙） | #FF8C42 | 紧急/警告、促销高亮 |
| 成功绿 | #52C41A | 成功状态、完成标记 |
| 警告黄 | #FAAD14 | 提示、等待状态 |
| 错误红 | #FF4D4F | 错误、SOS按钮、删除 |
| 正文黑 | #333333 | 主标题、正文 |
| 次级灰 | #666666 | 副标题、说明文字 |
| 辅助灰 | #999999 | 占位符、禁用态 |
| 分割线 | #E8E8E8 | 列表分隔、卡片边框 |
| 页面背景 | #F5F7FA | 页面底色 |
| 卡片背景 | #FFFFFF | 卡片、弹窗背景 |

### 1.3 字体规范

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 大标题 | 40rpx (20px) | Bold 700 | 1.4 | 页面标题 |
| H2 模块标题 | 34rpx (17px) | Semi-bold 600 | 1.4 | 模块标题 |
| H3 卡片标题 | 32rpx (16px) | Medium 500 | 1.4 | 列表项标题 |
| Body 正文 | 28rpx (14px) | Regular 400 | 1.6 | 正文内容 |
| Caption 辅助 | 24rpx (12px) | Regular 400 | 1.5 | 辅助说明、时间 |
| Mini 极小 | 20rpx (10px) | Regular 400 | 1.4 | 标签、角标 |

字体栈：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", Arial, sans-serif`

### 1.4 间距系统

基准单位：8rpx (4px)

| 名称 | 值 | 用途 |
|------|------|------|
| xs | 8rpx | 紧凑间距（图标与文字） |
| sm | 16rpx | 列表项内部 |
| md | 24rpx | 卡片内边距 |
| lg | 32rpx | 模块间距 |
| xl | 48rpx | 区域间距 |

### 1.5 圆角规范

| 元素 | 圆角值 |
|------|--------|
| 按钮 | 8rpx |
| 卡片 | 16rpx |
| 输入框 | 8rpx |
| 头像（圆形） | 50% |
| 弹窗 | 24rpx（顶部） |
| 标签 | 4rpx |

## 二、移动端组件库

### 2.1 导航栏 (NavigationBar)

```
┌─────────────────────────────────────┐
│  ← 返回    页面标题         ⋮ 更多  │
└─────────────────────────────────────┘
```

- 高度：88rpx（含状态栏）
- 背景：#FFFFFF 或渐变（主色→辅色）
- 标题：H2 居中，#333333
- 支持自定义右侧操作按钮

### 2.2 底部标签栏 (TabBar)

```
┌─────────────────────────────────────┐
│  🏠首页   📋订单   🆘求助   👤我的 │
└─────────────────────────────────────┘
```

- 高度：100rpx
- 图标：48rpx × 48rpx
- 选中色：主色 #4A90D9
- 未选中：#999999
- 各端标签配置不同，由 pages.json 定义

### 2.3 服务卡片 (ServiceCard)

```
┌─────────────────────────────────────┐
│ [图片]                               │
│ 服务标题                    ¥价格    │
│ 服务描述文字（最多2行）              │
│ ★4.8  |  已服务128次                │
└─────────────────────────────────────┘
```

- 卡片圆角：16rpx
- 内边距：24rpx
- 图片高度：200rpx
- 标题：H3，单行溢出省略
- 描述：Body，最多2行

### 2.4 订单卡片 (OrderCard)

```
┌─────────────────────────────────────┐
│ 订单号：YL20260601001      待服务    │
│─────────────────────────────────────│
│ 🏥 健康护理 - 日常陪护             │
│ 📅 2026-06-15 09:00                │
│ 📍 昆明市五华区                     │
│─────────────────────────────────────│
│ [查看详情]              [取消订单]   │
└─────────────────────────────────────┘
```

- 状态标签颜色：待处理(黄)、进行中(蓝)、已完成(绿)、已取消(灰)
- 操作按钮：右对齐，主操作用主色

### 2.5 SOS紧急按钮

```
    ┌─────────┐
    │         │
    │   SOS   │    红色圆形，直径 160rpx
    │         │    脉冲动画提示
    └─────────┘
```

- 背景：#FF4D4F，圆形
- 字号：48rpx Bold 白色
- 长按触发（防误触）
- 触发后震动反馈

### 2.6 聊天气泡 (ChatBubble)

```
┌──────────────────┐
│ 用户消息内容      │  → 蓝色背景白字，右对齐
└──────────────────┘

     ┌──────────────────┐
     │ AI回复内容        │  → 白色背景灰字，左对齐
     └──────────────────┘
```

- 用户气泡：背景 #4A90D9，文字 #FFFFFF
- AI气泡：背景 #F0F2F5，文字 #333333
- 最大宽度：70%
- 圆角：16rpx

### 2.7 列表项 (ListItem)

```
┌─────────────────────────────────────┐
│ [图标] 标题文字              箭头 → │
│        副标题/描述                   │
└─────────────────────────────────────┘
```

- 高度：自适应，最小 96rpx
- 左侧图标：40rpx × 40rpx
- 底部分割线：左缩进 88rpx

### 2.8 表单输入框 (FormInput)

```
标签名称
┌─────────────────────────────────────┐
│ 请输入内容...                        │
└─────────────────────────────────────┘
错误提示信息
```

- 高度：80rpx
- 边框：1rpx solid #E8E8E8
- 聚焦边框：主色 #4A90D9
- 错误边框：#FF4D4F

## 三、管理后台组件

### 3.1 布局结构

```
┌──────┬──────────────────────────────┐
│      │  顶部栏（Logo + 用户信息）   │
│ 侧   ├──────────────────────────────┤
│ 边   │                              │
│ 导   │      内容区域                │
│ 航   │                              │
│      │                              │
└──────┴──────────────────────────────┘
```

- 侧边栏宽度：240px（可折叠至 64px）
- 顶部栏高度：56px
- 内容区内边距：24px

### 3.2 数据表格 (DataTable)

- 表头背景：#FAFAFA
- 行高：48px
- 斑马条纹：奇数行 #FFFFFF，偶数行 #FAFBFC
- 操作列：最右侧固定
- 分页器：底部居中

### 3.3 统计卡片 (StatCard)

```
┌──────────────┐
│ 📊 今日订单  │
│    128       │
│  ↑12% 较昨日 │
└──────────────┘
```

- 标题：Caption 级别灰色
- 数值：36px Bold 主色
- 趋势：绿色上升/红色下降

### 3.4 标签页 (Tabs)

- 管理后台共11个标签页
- 选中态：下划线 + 主色文字
- 切换动画：0.3s ease

## 四、切图/标注规范

### 4.1 图标规格

| 用途 | 尺寸 | 格式 | 命名规范 |
|------|------|------|----------|
| TabBar图标 | 48×48 / 96×96 @2x | PNG | tab-{name}.png |
| 列表图标 | 40×40 / 80×80 @2x | PNG/SVG | icon-{name}.png |
| 功能图标 | 32×32 / 64×64 @2x | SVG | func-{name}.svg |
| 启动图 | 750×1334 | PNG | launch-{端}.png |
| Logo | 200×200 | PNG/SVG | logo.png |

### 4.2 图片资源规范

- 格式优先级：WebP > PNG > JPG
- 压缩：PNG 使用 tinypng，JPG 质量 80%
- 命名：全小写，连字符分隔
- 目录：`/assets/images/{模块名}/`

### 4.3 标注说明

所有设计稿标注使用 rpx 单位（移动端）或 px 单位（后台），标注内容包含：
- 元素尺寸（宽 × 高）
- 间距（margin / padding）
- 字体（字号 + 字重 + 颜色）
- 圆角值
- 阴影参数（如有）

### 4.4 适配说明

| 屏幕 | 处理方式 |
|------|----------|
| iPhone SE (375pt) | 最小适配宽度 |
| iPhone 14 (390pt) | 标准设计尺寸 |
| iPhone 14 Pro Max (430pt) | 大屏适配验证 |
| iPad | 不在MVP范围 |
| Android 主流 | uni-app rpx 自动适配 |
